# Staffetta Visualizer

* [Source Code](https://github.com/ybbarng/staffetta-visualizer)
* [Demonstration](https://staffetta.byb.kr)

## Prerequisites

### Install Dependencies
```
npm install
or
yarn install
```

### Instal gulp
```
npm install --global gulp
or
yarn global add gulp
```

## Build Client
```
gulp
```

## Run server
```
node index.js
```
